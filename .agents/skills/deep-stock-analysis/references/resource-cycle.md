# 资源周期适配器

## 价值驱动
储量品位、产量、实现价格、回收率、单位现金成本、矿山寿命及项目爬坡；区分价格顺风与运营改进。
## 领先指标
商品期货与库存、供需缺口、资本开支周期、品位与剥采比、项目投产进度、资源国税费政策。
## 壁垒与会计陷阱
成本曲线位置、勘探和复杂矿运营能力；关注资本化剥采、减值、复垦义务、并购对价和少数股东权益。
## 估值
中周期盈利、EV/EBITDA、P/NAV 与分矿山 DCF；压力测试价格、汇率、成本和产量，禁止只用峰值 PE。
## 风险
商品回落、事故停产、资源民族主义、项目延期、品位下降和高价并购。
## 证伪
同价环境下单位成本持续恶化、核心项目连续延期、储量替代率不足或自由现金流长期显著弱于利润。
