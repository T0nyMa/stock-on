# 股票分析方法论

## 研究核心与交易叠加层

本方法论分为两层：研究核心回答公司如何赚钱、质量如何、市场隐含什么预期以及何时被证伪；交易叠加层才处理时机、价位、仓位和风险控制。完整研究优先调用 `$deep-stock-analysis`，其 `research_summary` 可被场景复用。不得从研究结论直接跳到交易动作。

基于三花智控（002050）持仓分析的完整方法论沉淀。适用于有持仓的个股深度分析。

## Phase 1: 归因分析 — 跌的是什么？

### 目的

搞清楚股价波动是个股问题还是市场问题。这决定了是应该止损还是扛过去。

### 方法

**三步对比法**：

```
个股涨跌 = 大盘β + 板块α + 个股特异性
```

| 对比层级 | 参照物 | 要回答的问题 |
|----------|--------|-------------|
| 大盘 | 上证/深证/创业板 | 是不是系统性风险？大盘都在跌就别怪个股 |
| 板块 | 所属行业指数 | 是不是行业问题？板块都在跌就是行业β |
| 同行 | 3-5只同类个股 | 是不是个股问题？就你跌就是自身原因 |

### 三花案例

```
三花 -4.3% = 大盘(+0.9%) + 板块(-1.1%) + 个股特异性(-4.0%)
              ↑ 没问题     ↑ 拖累1/4      ↑ 主要矛盾
```

同行对比是关键一步：华域汽车 +0.4%，拓普集团 -7.2%，中鼎 -2.4%。分化越大，越说明是个股特异性问题（或细分主题问题，三花和拓普都是机器人标签重的）。

### 操作含义

| 归因结果 | 操作含义 |
|----------|---------|
| 大盘驱动（同步跌） | 系统性风险，控制总仓位，不着急补仓 |
| 板块驱动（板块都跌） | 行业β问题，关注行业拐点信号 |
| **个股驱动（就你跌）** | **分析自身原因，是估值贵了还是基本面坏了** |
| 个股+板块双重驱动 | 最复杂，需要分别判断两个因素的趋势 |

## Phase 2: 估值分析 — 底在哪里？

### 目的

找到股价的锚。技术面告诉你怎么走，估值告诉你能走到哪。

### 方法

**第一步：拆业务结构**

很多公司的业务是多元的，不同业务对应不同估值。必须拆开看：

```
三花智控 = 制冷(60%营收) + 汽零(40%营收) + 机器人(0%营收，纯概念)
           PE 12-18x     PE 15-25x      PE 无限（期权定价）
```

**第二步：算多层 PE 锚**

| 层 | PE | 含义 |
|----|-----|------|
| 铁底 | 行业最低估值 | 所有概念归零后的底线 |
| 合理 | 行业平均估值 | 给定当前信息的最公允定价 |
| 当前 | 实际PE | 市场正在给的价格 |
| 乐观 | 概念回归估值 | 催化剂兑现后的目标 |

```
三花案例：
  铁底：25x → 28元（机器人归零）
  合理：30-35x → 34-39元（机器人有小幅期权价值）
  当前：40x → 45元（机器人有期待但克制）
  乐观：50x → 56元（机器人故事回归）
  成本：52x → 58元（买入时定价了高预期）
```

**第三步：用估值解释价格**

亏的是估值压缩而不是业绩恶化：

```
买入价 57.9 = 52x PE
当前价 45.4 = 40x PE
差额 -12.5元 = -12x PE × 1.12 EPS

→ 亏在估值，不是亏在业绩。公司赚更多钱了(40→47亿)，但市场不愿给那么高PE了。
```

### 操作含义

| 估值判断 | 操作含义 |
|----------|---------|
| 接近铁底 | 不宜割肉，可考虑加仓 |
| 在合理区间 | 持有观望，等催化剂 |
| 高于合理 | 逢高减仓 |
| **当前：略高于合理，在压缩中** | **不止损也不加仓，做波段等催化** |

## Phase 3: 技术分析 — 现在该做什么？

### 3.1 多策略共识扫描

单一策略容易误判，多策略共振才有参考价值。

**策略选择原则**：不是所有策略都用，而是选跟当前市场状态匹配的。

| 市场状态 | 优先策略 | 不适用策略 |
|----------|---------|-----------|
| 多头趋势 | 均线金叉、放量突破、多头趋势 | 箱体震荡、地量 |
| 震荡盘整 | 箱体震荡、缠论、波浪理论 | 趋势类策略 |
| **空头/下跌** | **缩量回调、地量、一阳三阴、情绪周期** | 追涨类策略 |
| 题材驱动 | 热点题材、龙头股、事件驱动 | — |

**共识判断标准**：

```
7个策略中：
  ≥5个看多 → 强买入
  4个看多，3个持有/观望 → 偏多（三花当前状态）
  多方空方接近 → 震荡/观望
  ≥4个看空 → 减仓/回避
```

### 3.2 波浪理论定位

**目的**：判断价格在涨跌大结构中的位置。直接回答"跌完了吗"。

**关键判断**：

- 识别推动浪（1-2-3-4-5）还是调整浪（A-B-C）
- 找到斐波那契回撤关键位（38.2%、50%、61.8%、**78.6%**）
- **78.6% 回撤位是最后防线**，守住则结构完整，跌破则原趋势可能反转

```
三花案例：
  41.8→56.0 推动浪，56→当前 ABC调整
  78.6%回撤位 = 44.8 ← 恰好与A浪低点44.5共振
  → 这不是巧合，是技术上的强支撑确认
```

### 3.3 情绪周期定位

**目的**：判断市场心理阶段，找到极端点。

| 阶段 | 特征 | 操作 |
|------|------|------|
| 冰点 | 无人问津、缩量、RSI<30 | **逐步建仓** |
| 启动 | 开始放量、站上MA20 | 加仓 |
| 高潮 | 连阳加速、高换手、PE飙升 | **逐步减仓** |
| 退潮 | 暴跌、情绪逆转 | 减仓/止损 |

```
三花当前：退潮→冰点过渡
  特征：RSI6=31.8、量比0.71、换手率3%、无人关注
  意义：不是恐慌的时候，是耐心等冰点确认的时候
```

### 3.4 关键技术位确认

把技术支撑/阻力跟估值锚定结合起来，双重验证：

```
三花 44.5 = 箱体下轨 + A浪低点 + 78.6%斐波 + 40x PE
         ← 四重共振，可靠性高于单一技术位
```

## Phase 4: 决策框架 — 情景+触发+执行

### 目的

不预测方向，而是准备好所有可能性下的应对方案。

### 方法

**三情景框架**：

| 情景 | 概率 | 操作 | 触发条件 |
|------|------|------|---------|
| 基准情景 | 50-60% | 箱体波段 | 44.5-49.5区间 |
| 乐观情景 | 20-25% | 持有到目标 | 突破49.5+放量 |
| 悲观情景 | 20-25% | 止损 | 跌破44.5 |

**每个情景必须包含**：

- 明确的触发条件（价格、量能、时间）
- 具体的操作指令（买卖多少股）
- 仓位变化结果

```
错误示例："反弹就减仓"
正确示例："反弹到48.5-49.5且量比<1.2 → 卖出1000股 → 仓位降至3300股"
```

## 日常分析流程

### 首次深度分析

对一只新股票，按上方四层框架做完整分析，产出 `technical-analysis-report.md`，包含：

- 持仓现状
- 基本面定位（PE 锚定四层）
- 技术面共识（7 策略矩阵）
- 关键价位表（技术+估值双重验证）
- 波浪结构
- 情绪周期位置
- 题材热度
- 三情景操作方案

### 每日跟踪分析

**每次分析前必须先读 `technical-analysis-report.md`**，再执行当日分析。流程：

```
1. Read tracking/{code}/technical-analysis-report.md  ← 先读报告
2. 拉取最新数据（fetch + indicators）
3. 对比今日数据 vs 报告中的关键价位：
   - 是否触及止损线？
   - 是否触及卖出线？
   - 是否触及买入/加仓线？
4. 更新技术指标变化
5. 检查题材催化剂是否有进展
6. 判断情景切换（基准→乐观？基准→悲观？）
7. 生成当日 analysis.md
8. 更新 position.json
```

### 报告更新时机

不是每天重写，而是在以下情况更新：

| 触发条件 | 更新内容 |
|----------|---------|
| 关键价位被突破（止损/卖出/买入线） | 更新情景概率和操作建议 |
| 波浪结构确认变化（如C浪结束） | 重新标记浪型 |
| 题材出现新催化剂 | 更新题材热度和目标价 |
| 财报/重大公告 | 更新估值锚定 |
| 每 2 周 | 全面重审 |

## 执行清单

### 首次深度分析

分析一只持仓股票时，逐项检查：

### 数据层
- [ ] 最新 K 线+实时行情（fetch.py）
- [ ] 最新技术指标（indicators.py）
- [ ] 近 1 月 K 线明细
- [ ] 近 3 月月度涨跌

### 归因层
- [ ] 大盘 3 日/5 日涨跌对比
- [ ] 板块 3 日/5 日涨跌对比
- [ ] 3-5 只同行 3 日涨跌对比
- [ ] 归因结论：大盘/板块/个股各占多少

### 估值层
- [ ] 拆解业务结构（不同业务×不同PE）
- [ ] 计算铁底/合理/当前/乐观四层PE锚
- [ ] 判断当前在哪一层

### 技术层
- [ ] 7 策略共识扫描（至少跑 5 个）
- [ ] 波浪结构定位（涨跌大结构+斐波位）
- [ ] 情绪周期阶段判断
- [ ] 关键价位三重共振验证

### 决策层
- [ ] 三情景（基准/乐观/悲观）+ 概率
- [ ] 每个情景的触发条件（价格/量能）
- [ ] 每个情景的具体操作（买卖数量）
- [ ] 仓位变化结果
