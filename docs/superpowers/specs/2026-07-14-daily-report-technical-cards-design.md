# 日报技术分析卡设计

## 目标

将 2026-07-14 日报的技术结论结构化，使每个结论可追溯到同一组登记数据字段，并让核心股与观察股可以横向比较。

## 范围

- 核心五股：工业富联、山东黄金、三花智控、兆易创新、阿里巴巴。
- 其余 17 只追踪股。
- 仅改写登记日报 `tracking/daily/positions/2026-07-14.md` 及其对应 HTML；不修改交易记录或持仓成本。

## 内容结构

### 核心技术分析卡

每只核心股在第三章使用相同顺序展示：

1. 多周期趋势：日、周、月状态和一致性。
2. 均线与动量：MA5/10/20、MACD、RSI6/12。
3. 强度与资金：ADX、MFI、CMF、可用时的量比与换手。
4. 关键价位：支撑、阻力、观察区、失效位、目标和风险收益比。
5. 结构化结论：趋势状态、是否满足入场确认、持仓或观察动作。

有 A/H 两地价格的股票额外展示 H 股技术字段与按当日 HKD/CNY 折算的价格差；无可用字段时明确标记，不推算。

### 观察股指标矩阵

第五章保留“较强 / 中性 / 偏弱”分组；每一组表格增加：

- 收盘与涨跌幅；
- 日、周、月趋势；
- MA5/10/20；
- MACD；
- RSI6/12；
- ADX；
- MFI/CMF；
- 观察区、失效位、目标和 RR；
- 一句行动结论。

## 数据与缺失处理

- A 股字段只取 `snapshot.quote`、`snapshot.indicators`、`artifact.report_context` 与当日技术快照。
- 港股技术字段只取 `data/market/hk_snapshot.json` 和 `data/market/hk_klines.json` 的当前日线计算结果。
- 不以旧策略扫描冒充当日结论；策略聚合与技术结论分列。
- 任一字段不可用时显示 `unavailable` 与原因；不补零、不估算。

## 验收

- 22 只追踪股各出现一次且只出现一次。
- 五张核心技术卡字段顺序一致。
- 观察股矩阵每行包含趋势、均线、MACD、RSI、ADX、资金/量价、关键价位和结论。
- Markdown 与 HTML 均保留七章结构，且 HTML 从 Markdown 重新渲染。
