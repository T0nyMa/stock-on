# 全市场股票筛选器

从全市场 5000+ 只 A 股中，按多个维度筛选热门股和潜力股，发现追踪池之外的候选。

数据源：新浪财经实时行情（通过 akshare `stock_zh_a_spot`），**单日快照**（含开/高/低/收/量，不含历史K线）。

## 用法

`/screener` 或用户说"筛选"/"热门股"/"有什么好股票"/"潜力股"

## 流程

### 1. 运行筛选

```bash
source .venv/bin/activate && python src/screener.py
```

耗时约 20-30 秒，拉取 5000+ 只股票实时行情。

输出：`data/market/screener.json`

### 2. 读取结果

Read `data/market/screener.json`，三大类 10 个维度：

| 类别 | 维度 | 逻辑 |
|------|------|------|
| 🔴 机会 | `mild_breakout` | 涨幅3-7%+放量+收于中上部，最值得跟踪 |
| 🔴 机会 | `breakout` | 涨幅2-6%+光头阳线，买方主导突破 |
| 🔴 机会 | `gap_up` | 跳空高开+日内未补缺口 |
| 🟢 超跌 | `deep_cut` | 跌5-9%未跌停，超卖反弹机会 |
| 🟢 超跌 | `stablizing` | 跌幅收窄+振幅缩小，接近底部 |
| 🟢 超跌 | `panic_sell` | 放量下跌中，等止跌确认 |
| 🔵 异动 | `heavy_volume` | 高振幅+大成交，多空激烈 |
| 🔵 异动 | `doji` | 宽幅十字星，即将变盘 |
| 🔵 异动 | `top_amount` | 全市场成交额TOP30 |
| 🔵 异动 | `wild_swing` | 极端振幅>8% |

### 3. 交叉对比追踪池

Read `tracking/tracklist.json`，从筛选结果中**排除已追踪的股票**。

重点关注：
- 出现在多个维度中的股票（如同时出现在涨幅榜和成交额榜）
- 与追踪股票**同板块**的新面孔（通过名称/行业推断）
- **成交额榜前10**但不在追踪池的

### 4. 输出发现报告

Write `tracking/daily/discovery/YYYY-MM-DD.md`

格式：
```markdown
# 异动发现 — YYYY年MM月DD日

## 多维度共振（最值得关注）
| 股票 | 代码 | 出现维度 | 涨幅 | 成交额 | 板块 | 理由 |

## 与追踪池同板块的新面孔
| 股票 | 板块 | 关联追踪股 | 涨幅 | 理由 |

## 成交额榜（市场主线）
（前10，标注已追踪/新面孔）

## 建议关注
（2-3只最值得深度分析的股票，说明理由）
```

### 5. 对候选股快速初筛

对报告中的 2-3 只最值得关注的候选，用现有 fetch + indicators 拉数据，做一次快速技术面判断（趋势+RSI+乖离），给出"值得建仓分析"还是"等回调再看"的结论。

## 数据说明

- **数据范围**：沪深 A 股（不含北交所、ST、新股）
- **数据时效**：单日快照（开/高/低/收/量），不含历史K线
- **局限**：无法计算 MA/MACD/RSI 等趋势指标（需额外 fetch 单股 K 线）
